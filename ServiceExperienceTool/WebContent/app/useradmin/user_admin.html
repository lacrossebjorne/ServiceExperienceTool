SET användaradministration

Använd menyn för att administrera användare och roller.
  • Lägg till användare: lägg till användare i Service Experience Tool.
  • Lista användare: Lista av alla användare i systemet, lägg till/ta bort användare samt information om användare.
  • Administrera roller: Lista alla roller i systemet, lägg till/ta bort roller och se användare som är tilldelad en roll.
